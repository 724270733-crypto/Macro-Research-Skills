# 量化研报复现工具

这个skill用于将金融量化研报（已从PDF转换为Markdown格式）转换为AI coding能够理解和执行的完整代码实现。

## 使用场景

当用户需要：
- 将量化策略研报转换成可运行的Python代码
- 详细分析研报中的方法、公式和步骤
- 复现研报中的回测结果
- 理解策略的完整实现逻辑

## 输入要求

用户提供：
1. 研报的Markdown文件路径
2. （可选）指定需要特别关注的策略/因子/模型

## 输出格式

生成一个完整的复现指南，包含：

### 1. 方法论总结 (Methodology Summary)
用简洁的语言描述：
- 策略的核心思想
- 使用的关键算法
- 数据来源和处理方式
- 预期的输出结果

### 2. 详细步骤说明 (Step-by-Step Instructions)

按执行顺序列出每个步骤：
```
## 步骤1: 数据获取
### 目标
[说明这个步骤要做什么]

### 输入
- 数据源：[如 Tushare / Wind / 自定义CSV]
- 参数：[如 股票池、时间范围、字段]

### 处理逻辑
[用自然语言描述算法逻辑]

### 关键公式
用LaTeX格式写出数学公式

### 注意事项
- 可能的坑
- 参数敏感性
```

## 处理流程

1. **读取研报内容**：解析Markdown文件
2. **提取关键信息**：
   - 策略类型（选股/择时/配置/套利等）
   - 核心算法和公式
   - 数据来源和处理方法
   - 回测参数和设置
   - 结果评估指标
3. **结构化输出**：按照上述格式组织内容
4. **代码生成**：编写完整的Python实现代码

## 注意事项

- 如果研报中缺少某些实现细节，需要明确标注"需要用户补充"
- 对于不确定的参数，提供合理的默认值并说明
- 如果涉及API调用，需要说明所需的API权限和数据
- 代码应包含详细的注释，便于理解和修改

- `ifind-data-skill/` - 同花顺iFind金融数据查询
- `market-environment-analysis/` - 全球市场环境分析
- `market-news-analyst/` - 市场新闻事件分析
- `china-macro-analyst/` - 中国宏观经济与政策分析
- `cross-border-flow-tracker/` - 沪深港通资金流向追踪
- `sector-analyst/` - 行业板块分析
- `backtest-expert-0.1.0/` - 量化策略回测指导
- `quant-research-reproducer/` - 研报复现与代码生成
- `macro-weekly-report/` - 宏观策略周报/日报生成

### 1. market-environment-analysis（全球市场环境分析）

**来源**：`market-environment-analysis/SKILL.md`

**功能**：综合分析全球市场环境，包括美股、欧股、亚洲市场、外汇、商品期货、经济指标等，提供风险偏好评估、技术指标解读等专业分析报告。

**核心能力**：
- 收集最新市场数据：标普500、纳斯达克、道琼斯、日经225、上证综指、恒生指数等
- 外汇汇率：USD/JPY、EUR/USD 等主要货币对
- 商品价格：WTI 原油、黄金、白银
- 美国国债收益率：2年、10年、30年期
- VIX 指数（恐慌指标）
- 风险偏好评估（Risk-on/Risk-off）
- 行业板块表现分析
- 技术指标解读

**报告结构**：
```

2. 全球市场概览（美国、亚洲、欧洲市场）
3. 外汇与商品趋势
4. 关键事件与经济指标
5. 风险因素分析
6. 投资策略含义
```

**分析示例**：

- Japan GDP Flash

### 2. market-news-analyst（市场新闻事件分析）

**来源**：`market-news-analyst/SKILL.md`

**功能**：分析近10天重大市场新闻事件对股市和商品市场的影响，通过结构化的6步工作流生成专业的英文分析报告。

**核心能力**：
- 央行货币政策评估（FOMC、ECB、BOJ等）
- 通胀数据解读（CPI、PPI、PCE）
- 就业数据（NFP、失业率、工资）
- 地缘政治事件对大宗商品影响分析
- 巨型市值公司（Mega-cap）财报发布解读
- 历史事件模式比对

**分析框架（6步工作流）**：
1. **新闻收集**：通过 WebSearch/WebFetch 收集近10天新闻
2. **加载知识库**：加载市场事件模式、新闻源可信度等参考文档
3. **影响程度评估**：计算影响评分 `Impact Score = (价格影响分 × 影响范围倍数) + 前瞻性调整`
4. **市场反应分析**：分析即时反应、多资产响应、模式比对
5. **相关性分析**：评估事件间相互作用（增强、抵消、序列效应）
6. **报告生成**：按影响程度排序生成英文 Markdown 报告

**影响评分维度**：
- 价格影响：严重(±2%+)、主要(±1-2%)、中等(±0.5-1%)、次要(±0.2-0.5%)
- 影响范围：系统性(3倍)、跨资产(2倍)、行业范围(1.5倍)、个股(1倍)
- 前瞻性：制度变化(+50%)、趋势确认(+25%)、孤立事件(0%)、反向信号(-25%)

**触发关键词**：`分析新闻`、`市场动态`、`最近有什么大事`、`财报分析`

### 3. china-macro-analyst（中国宏观经济分析）

**来源**：`china-macro-analyst/SKILL.md`

**功能**：分析中国宏观经济数据、央行货币政策、财政政策及其对A股和港股市场的影响，解读政策信号并评估市场影响。

**关键经济指标**：
| 类别 | 指标 |
|------|------|
| 增长指标 | GDP增速、工业增加值、社会消费品零售总额、固定资产投资 |
| 价格指标 | CPI、PPI、核心CPI |
| 活动指标 | 制造业PMI、非制造业PMI、财新PMI |
| 金融指标 | 社会融资规模、M2增速、新增人民币贷款、LPR |
| 外部指标 | 出口、进口、贸易顺差、外汇储备、人民币汇率 |

**货币政策工具**：
- **利率工具**：1年LPR、5年LPR、MLF利率、逆回购利率、存款准备金率
- **数量工具**：逆回购、MLF、国库定存

**政策信号解读**：
- 降准：释放长期流动性 → 利好银行、地产
- 降息：降低融资成本 → 利好成长股
- 结构性工具：定向支持特定领域（科技、绿色、小微）

**市场影响矩阵**：
| 经济事件 | A股影响 | 港股影响 | 受益板块 | 受损板块 |
|----------|---------|---------|---------|---------|
| 降准 | + | + | 银行、地产 | - |
| 降息 | + | + | 成长股 | 银行 |
| CPI上行 | - | - | 消费 | 成长股 |
| PMI超预期 | + | + | 周期股 | 防御股 |

**数据来源**：国家统计局(stats.gov.cn)、人民银行(pbc.gov.cn)、海关总署(customs.gov.cn)、财政部(mof.gov.cn)

**触发关键词**：`中国宏观经济`、`央行政策`、`经济数据`、`政策信号`

### 4. cross-border-flow-tracker（沪深港通资金流向追踪）

**来源**：`cross-border-flow-tracker/SKILL.md`

**功能**：追踪沪深港通机制下跨境资金流动，监控北向资金和南向资金动向，识别趋势个股和机构持仓分析。

**北向资金（沪股通+深股通）**：
- 每日额度：520亿元人民币
- 合格股票：主要指数成分股、大中盘股

**南向资金（港股通）**：
- 每日额度：420亿元人民币
- 合格股票：恒生指数/恒生国企指数成分股、大中盘股

**核心能力**：
- 北向资金实时追踪
- 南向资金实时追踪
- 趋势个股识别
- 机构持仓分析
- 行业配置分析

**信号解读**：
- **看多信号**：北向资金连续大幅净流入（>50亿/日持续3天以上）、核心资产获持续加仓、外资持股比例创新高、流入与A股上涨同步
- **看空信号**：北向资金持续净流出、重仓股被大幅减持、流出伴随指数下跌、外资对某行业系统性减仓
- **背离信号**：市场下跌但北向持续流入（可能底部）、市场上涨但北向持续流出（可能顶部）

**数据来源**：东方财富(eastmoney.com)、同花顺(10jqka.com.cn)、港交所(HKEX)、Wind资讯(wind.com.cn)

**触发关键词**：`北向资金`、`南向资金`、`外资动向`、`Stock Connect`、`沪港通/深港通`

### 6. sector-analyst（行业板块分析）

**来源**：`sector-analyst/SKILL.md`

**功能**：分析行业和板块表现图表，评估市场定位和轮动模式，识别当前市场周期阶段并预测下一阶段情景。

**分析工作流（5步）**：
1. **数据收集和观察**：从图表提取行业表现数据，比较1周 vs 1个月表现识别趋势一致性
2. **市场周期评估**：对照典型周期模式（早期复苏、中期扩张、晚期、衰退）识别当前所处阶段
3. **当前情况分析**：客观评估表现模式，识别确认信号和矛盾信号
4. **情景发展**：基于轮动原则开发2-4个潜在情景，分配概率
5. **输出生成**：结构化 Markdown 文档

**输出格式**：
```markdown
# Sector Performance Analysis - 日期

## Executive Summary

## Current Situation
### Market Cycle Assessment
### Performance Patterns Observed

## Supporting Evidence
### Confirming Signals
### Contradictory Signals

## Scenario Analysis
### Scenario 1: [名称] (概率: XX%)
### Scenario 2: [名称] (概率: XX%)

## Recommended Positioning
### Strategic Positioning (中期)
### Tactical Positioning (短期)

## Key Risks
```

**概率指南**：
- 70-85%：强证据，多个确认信号
- 50-70%：中等证据，部分确认信号但指标混合
- 30-50%：弱证据，有限或冲突信号
- 15-30%：投机性情景

**触发关键词**：`板块分析`、`行业轮动`、`市场定位`、`板块表现图表`

### 8. backtest-expert-0.1.0（量化策略回测指导）

**来源**：`backtest-expert-0.1.0/SKILL.md`

**功能**：系统化回测交易策略的专业指导，强调策略稳健性而非回测利润，包含完整的方法论和压力测试框架。

**核心理念**：寻找"最不容易崩溃"的策略，而非"回测利润最高"的策略。

**核心原则**：增加摩擦、压力测试假设、观察什么能存活。如果策略在悲观条件下仍能存活，它更有可能在实际交易中表现良好。

**回测工作流（6步）**：
1. **陈述假设**：用一句话定义策略优势
2. **编写规则**：零裁量，完全规则化明确定义入场、出场、仓位管理、过滤器
3. **运行初始回测**：至少5年（最好10年+），多个市场环境
4. **压力测试策略**：
   - 参数敏感性：测试止损在50%-150%基准值下的表现，寻找"稳定平台"而非"狭窄峰值"
   - 执行摩擦：滑点1.5-2倍典型估计，最坏情况成交
   - 时间稳健性：逐年分析，要求大多数年份正期望
   - 样本大小：绝对最小值30笔交易，首选100+笔
5. **样本外验证**：Walk-forward分析
6. **评估结果**：决定部署✅ / 优化🔄 / 放弃❌

**关键测试原则**：
- **惩罚策略**：佣金高于实际、滑点1.5-2倍、最坏情况成交、部分成交
- **寻找平台而非峰值**：策略在止损1.5%-3.0%范围内均盈利才是好的
- **测试所有案例而非精选**：包括失败的股票，避免幸存者偏差
- **分离想法生成与验证**：直觉得到假设，验证必须纯数据驱动

**触发关键词**：`回测`、`策略验证`、`稳健性测试`、`避免过拟合`、`系统化交易`

### 9. quant-research-reproducer（研报复现）

**来源**：`quant-research-reproducer/SKILL.md`

**功能**：将金融量化研报（Markdown格式）转换为AI可理解的代码和详细复现步骤，生成可直接运行的Python代码和说明文档。

**输入**：用户提供的研报Markdown文件路径及（可选）重点关注的策略/因子/模型

**输出格式（6部分）**：
1. **方法论总结**：核心思想、关键算法、数据来源和处理方式
2. **详细步骤说明**：每个步骤的目标、输入、处理逻辑、关键公式（LaTeX格式）
3. **完整代码实现**：结构良好的Python脚本
4. **代码详解**：每个代码段的功能说明表格
5. **数据需求说明**：数据源、字段要求、格式要求、预处理注意事项
6. **可能的改进方向**：参数优化空间、风险控制增强点、扩展策略方向

**代码结构标准模板**：
```python
"""
策略名称
基于研报名称
参考来源: 研报出处
"""

# ============================================================
# 第一部分：配置参数
# ============================================================
class Config:
    START_DATE = "2020-01-01"
    END_DATE = "2025-12-31"
    LOOKBACK_PERIOD = 20
    HOLDING_PERIOD = 5

# ============================================================
# 第二部分：数据获取
# ============================================================
def fetch_data(...) -> pd.DataFrame: ...

# ============================================================
# 第三部分：因子计算
# ============================================================
def calculate_factors(...) -> pd.DataFrame: ...

# ============================================================
# 第四部分：信号生成
# ============================================================
def generate_signals(...) -> pd.DataFrame: ...

# ============================================================
# 第五部分：回测引擎
# ============================================================
def backtest(...) -> dict: ...

# ============================================================
# 第六部分：主程序
# ============================================================
def main():
    raw_data = fetch_data(...)
    processed_data = preprocess_data(raw_data)
    factors = calculate_factors(processed_data)
    signals = generate_signals(factors)
    results = backtest(signals, processed_data)
    print(f"年化收益: {results['annual_return']:.2%}")

if __name__ == "__main__":
    main()
```

**触发关键词**：`复现研报`、`把研报转成代码`、`分析这个量化策略`
